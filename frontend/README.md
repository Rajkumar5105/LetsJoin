# LetsJoin – Frontend

Frontend for **LetsJoin**, a sports turf booking and team coordination platform.

Built using React + Vite with a modern, responsive, dark-themed UI.

##  Tech Stack

- React (Functional Components + Hooks)
- Vite
- Tailwind CSS
- React Router DOM
- Axios
